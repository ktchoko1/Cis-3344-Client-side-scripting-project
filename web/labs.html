Lab 5

JavaScript Framework, where I learn how to create a simple framework that offers HTML code reuse. The HTML coder design their complete layout in a "template file" (file that has a placeholder for content) then they create several HTML "snippet files" that contain their content. When the user clicks on a nav link, the framework code read (using AJAX) the appropriate HTML snippet file and place its content into the content placeholder. This template technique is like how Visual Studio Dot Net implements HTML code reuse using what they call a "master file".
